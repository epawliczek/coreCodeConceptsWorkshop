Advanced Time Series Visualizations¶

Time series data can be an advanced concept, it is important, but will largely be handled by applications rather than code unless a regular report.

The following example is only to illustrate the complexity of time series and why many people give up on coding visualizations.

# Importing time package to convert string type date into 
import time


def timeStamp(timeString, timeFormat = '%Y-%m-%d'):
    timeStp = time.mktime(time.strptime(timeString, timeFormat))
    return timeStp


# Adding new columns for timestamp
# Uses 'apply' to send function across every row at once, required for distributed process, e.g. HADOOP

customerOrdersNoNADf = customerOrdersNoNADf.assign(\
                            shippedDateTs = customerOrdersNoNADf['shippedDate'].apply(lambda x: timeStamp(x)),\
                            requiredDateTs = customerOrdersNoNADf['requiredDate'].apply(lambda x: timeStamp(x)),\
                            index = customerOrdersNoNADf.index)
